This is the shared folder specifically for fitting the hBN/ET data from the N7 session.

The fit results should be stored in either a csv file or a excel file, and the figure results should be saved in a powerpoint file or similar.

ET_fitting_profile is a matlab file that has a basic least squares fit 

PhP fitting reference has the fitting equation
The reference files for this are "fitting_profile_hyp.m" with the data file "profile_hyp2.dat"

The data is stored in S3_mat.csv and xs.csv, with the x values shared among all the curves. S3_mat contains the line cuts of the S3 signal per frequency.
